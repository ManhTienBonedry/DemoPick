PHAM VI DU AN

- Uu tien giu nguyen nghiep vu dang chay on dinh; chi toi uu theo tung lop.

QUY TAC CHON DESIGNER VS DYNAMIC
- NEN DE DESIGNER (keo-tha) khi:
  - Cac thanh phan khung tinh: Form chinh, Sidebar, Header, Footer
  - Cac layout panel co cau truc co dinh: GroupBox, SplitContainer, ToolStrip, StatusStrip
  - Cac dialog CRUD don gian (Form them/sua co truong co dinh)

- NEN CODE DYNAMIC khi:
  - Danh sach lap lai theo du lieu: card san pham, booking block, timeline, list item tuy bien
  - UI phu thuoc vai tro/trang thai: an-hien nut, doi mau theo status, badge canh bao
  - Animation/chuyen trang/chuyen trang thai
  - Cac widget can cap nhat realtime (so lieu KPI, countdown, activity feed)

- NEN HYBRID (khuyen nghi manh):
  - Designer tao khung + placeholder panel
  - Code dong sinh control vao placeholder panel
  - Chuan hoa bang helper/component tai su dung

GOI Y PHAN LOAI CHO DU AN HIEN TAI
- Designer uu tien:
  - FrmChinh (khung tong)
  - FrmAuthHost / FrmDoiMatKhau / FrmDoiCaBooking / FrmThemSP (dialog truong co dinh)

- Dynamic uu tien:
  - UCDatLich (timeline ve booking, zoom, status color, note)
  - UCBanHang (catalog card, cart line, tang/gi am so luong, filter)
  - UCThanhToan (tong tien theo payment state, canh bao thao tac)
  - UCTongQuan (feed hoat dong, KPI cap nhat)

- Hybrid:
  - UCKhachHang, UCKhoHang, UCBaoCao (khung Designer + luoi/list render dong)

MUC TIEU HIEU NANG DE CAM NHAN MUOT
- FPS cam nhan: 55-60 cho animation ngan
- Phan hoi thao tac chinh: < 100ms
- Chuyen module: < 300ms voi du lieu cache
- Truy van nang/bao cao: async + loading skeleton, khong khoa UI thread
